== Description ==
Price as per unit plugin takes the price of the product and divides this price by its weight, which help to sell products to the customer to know how much a weight unit costs.
For example when selling any items. Price of goods/items calculated per weight and then shown according to customer like.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the /Plugins/ screen in WordPress
3. Woocommerce->settings->Price per unit for wc product
Note: This plug-in works only if a weight set on the product. This is an independent setting used for shipping purposes.

== Features of Plugin ==
Helps to determine how much unit cost.
Takes the product price from product.
Calculate the Unit cost divide by Product weight.
